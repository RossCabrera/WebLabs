# ğŸ¾ TinDog Project Guide

This project is a landing page for a dog-based dating application. It is designed to practice responsive web design using **Bootstrap 5**, focusing on grid layouts, components, and utility classes.

---

## ğŸ—ï¸ Project Components & Requirements

To successfully build this project, the following sections must be implemented:

1. **Navigation Bar**: A responsive Bootstrap `navbar` with links for Contact, Pricing, and Download.
2. **Hero Section**: A split-screen layout (`row` and `col`) featuring the main heading and mockup image.
3. **Features**: A 3-column grid layout using the provided SVG icons.
4. **Testimonial Carousel**: A sliding carousel component for user reviews.
5. **Press Section**: A row of partner logos (optional).
6. **Pricing Section**: A responsive card deck (`card-group` or `row/col`) showing subscription plans.
7. **Call to Action (CTA)**: A final section with download buttons.
8. **Footer**: Social icons and copyright information.

> [!IMPORTANT]
> **HINT:** To figure out which Bootstrap components or snippets are used, refer to the official [Bootstrap Snippets PDF](https://github.com/appbrewery/tindog/blob/main/Bootstrap-snippets.pdf).
