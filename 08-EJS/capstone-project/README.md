# 📝 Capstone Project 3: Blog Web Application

## 📌 Project Overview

This project consists of building a **Blog Web Application** using **Node.js**, **Express.js**, and **EJS**. The application allows users to create, view, edit, and delete blog posts through a clean and intuitive interface ✨.

The main focus of this project is learning **server-side rendering**, **routing**, and **templating**, while also paying close attention to **styling** and **responsiveness** 📱💻. Blog posts are stored in memory, meaning they **do not persist between sessions**, as no database is used in this version.

Design inspiration:

- 🔗 [Paul Graham Articles](http://www.paulgraham.com/articles.html)

---

## 🎯 Goals

- 🚀 Learn how to build a web server using Node.js and Express.js  
- 🧩 Understand server-side rendering with EJS  
- 🛠️ Practice CRUD operations (Create, Read, Update, Delete) without a database  
- 🎨 Improve CSS styling and responsive design skills  
- 😊 Deliver a simple and pleasant blogging experience  

---

## 📦 Deliverables

- 📁 One Node.js project containing the full website functionality  
- 🧱 At least one **EJS** file for the website structure  
- 🎨 At least one **CSS** file for styling  
- 🌐 A working Express.js server handling all routes  

---

## ✅ Required Features

### ✍️ Post Creation

- Users can create new blog posts using a form  
- Each post includes a **title** and **content**

### 👀 Post Viewing

- The home page displays all created blog posts  
- Posts are shown in a clean and readable format  

### ✏️ Post Update & 🗑️ Delete

- Users can edit existing blog posts  
- Users can delete posts when no longer needed  

### 🎨 Styling & 📱 Responsiveness

- The application must be well-styled using CSS  
- The layout should adapt properly to mobile and desktop screens  

---

## ⚙️ Technical Requirements

- 🟢 **Node.js**  
  Used as the runtime environment for the application  

- 🚏 **Express.js**  
  Handles routing, middleware, and server logic  

- 🧩 **EJS (Embedded JavaScript Templates)**  
  Used to render dynamic HTML pages  

- 🎨 **CSS**  
  Used for layout, styling, and responsiveness  
  *(Flexbox, Grid, or Bootstrap may be used)*

## 🚀 Getting Started

```bash
cd blog-app
bun install
bun server.js
```

Visit `http://localhost:3000` to use the app. Posts are stored in memory and reset on server restart.
