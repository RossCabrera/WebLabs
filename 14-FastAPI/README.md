# 14 — FastAPI

FastAPI for building modern Python APIs.

## Topics
- Path operations, request/response models
- Pydantic validation
- Dependency injection
- Async support
- OpenAPI / Swagger docs

## Structure
```
14-FastAPI/
├── notes/        ← concept notes per topic
└── exercises/    ← practice endpoints and mini-apps
```

## Setup
```bash
mise install        # activates pinned Python version
uv sync             # installs dependencies
uvicorn main:app --reload
```
