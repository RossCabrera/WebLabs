JavaScript FAQs

1. What is JavaScript? add
JavaScript is a programming language used to make web pages interactive.
2. What is jQuery? add
jQuery is a JavaScript library that simplifies DOM manipulation, events, and animations.
3. What is the DOM? add
The Document Object Model (DOM) is a representation of a web page that allows scripts to access and manipulate its content and structure.
4. What is an event in JavaScript? add
An event is an action or occurrence, like a click or key press, that can be detected and handled by JavaScript.
5. What is the difference between `.text()` and `.html()` in jQuery? add
`.text()` sets or gets plain text content, while `.html()` can include HTML elements.
6. How do you hide or show elements using jQuery? add
You can use `.hide()`, `.show()`, `.toggle()`, `.fadeIn()`, `.fadeOut()`, or `.slideToggle()`.
7. What is a callback function in JavaScript? add
A callback is a function passed as an argument to another function, which is executed after the first function completes.
8. What does `$(this)` refer to in jQuery? add
It refers to the current element that triggered the event.
9. How do you add a class to an element with jQuery? add
You can use `.addClass("className")` to add a class to the selected element.
10. Can jQuery and vanilla JavaScript be used together? add
Yes, you can use both together. jQuery is just a library that simplifies JavaScript, so they are fully compatible.
